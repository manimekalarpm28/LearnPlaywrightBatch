# VS CODE SHORTCUTS FOR WINDOWS

## General Shortcuts
| Action | Shortcut |
|--------|----------|
| Command Palette | `Ctrl+Shift+P` / `F1` |
| Quick Open (Go to File) | `Ctrl+P` |
| Go to Line | `Ctrl+G` |
| Go to Symbol in File | `Ctrl+Shift+O` |
| Go to Symbol in Workspace | `Ctrl+T` |
| Show All Commands | `Ctrl+Shift+P` |

## File Operations
| Action | Shortcut |
|--------|----------|
| New File | `Ctrl+N` |
| Open File | `Ctrl+O` |
| Save | `Ctrl+S` |
| Save All | `Ctrl+K S` |
| Save As | `Ctrl+Shift+S` |
| Close Editor | `Ctrl+W` |
| Close Folder | `Ctrl+K F` |
| Close Window | `Alt+F4` |

| Action | Shortcut |

| Focus Group Below | `Ctrl+K Ctrl+Down` |

| Go to Previous Editor Group | `Ctrl+PageUp` |
